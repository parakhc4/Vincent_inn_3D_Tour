Project: Vincent Inn 3D Virtual Tour
Developer: KGMediaWorks

To view the tour locally:
1. Unzip the folder.
2. Open a terminal in this folder.
3. Run: python3 -m http.server 8000
4. Open browser: http://localhost:8000

All files are unencrypted. Please do not redistribute or modify without permission.

- Contains scenes: Junior Suite, Lobby, Cafe, etc.
- Custom music and sidebar navigation included
